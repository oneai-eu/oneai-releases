# v0.3.0-beta — Developer Changelog

**Release Date:** 2026-02-26
**GitHub Release:** [v0.3.0-beta](https://github.com/0codekit/oneai/releases/tag/v0.3.0-beta)

## Breaking Changes

- **Chats are now associated with spaces instead of projects. The projects table has been dropped and replaced by spaces with provider='project'. Migration automatically converts existing projects to project-spaces and backfills chat ownership.**
- **User API keys moved from single-key column on users table to dedicated user_api_keys table supporting multiple keys. Existing keys are automatically migrated.**
- **Sonnet 4.5 output pricing corrected from $6/MTok to $15/MTok (60% increase). This matches Anthropic's official pricing.**

## Feature

- Die gesamte Chat-Architektur wurde umgebaut: Chats sind jetzt direkt mit Spaces verknüpft statt mit Projekten. Das vereinfacht die Struktur und ermöglicht flexiblere Arbeitsabläufe.
- Laufende KI-Antworten können jetzt gestoppt werden. Bei Verbindungsabbrüchen setzt der Chat nahtlos dort fort, wo er aufgehört hat.
- Ihr könnt jetzt mehrere API-Schlüssel erstellen und verwalten — jeder mit eigener Beschreibung und optionalem Ablaufdatum.
- Die neuesten Anthropic-Modelle stehen euch zur Verfügung — Opus 4.6 für komplexe Aufgaben, Sonnet 4.6 als leistungsstarke und kostengünstige Alternative.
- Screenshots und Bilder können jetzt direkt aus der Zwischenablage per Strg+V in den Chat eingefügt werden.
- Wenn die Verarbeitung eines Dokuments fehlschlägt, wird sie automatisch erneut versucht — mit zunehmenden Wartezeiten, um das System nicht zu überlasten.
- Die Grundlage für KI-Bildgenerierung ist gelegt — die API unterstützt jetzt Bildgenerierung über OpenAI-kompatible Modelle.
- Artefakte zeigen jetzt an, wer sie erstellt hat und wann sie zuletzt bearbeitet wurden. Außerdem können Artefakte zwischen Projekten verschoben oder kopiert werden.
- Eine neue Seite zeigt euch alle bisherigen Updates und Änderungen von OneAI — inklusive automatischer Benachrichtigung bei neuen Releases.
- Neue Startseite nach dem Login, Massenlöschung für Artefakte und Chats, und eine aufgeräumte Admin-Navigation.
- Der weclapp-Connector erstellt jetzt automatisch verständliche Zusammenfassungen eurer Geschäftsdaten und reichert sie mit Kundenkontext an.
- Der weclapp-Connector bietet jetzt gezieltere Datenauswahl und übersichtlichere Ordnerstrukturen.

## Ui

- Alle laufenden Aktivitäten — Uploads, Dokumentenverarbeitung und minimierte Spaces — findet ihr jetzt übersichtlich in der Sidebar statt als schwebende Icons.
- Reasoning-Schritte und Tool-Aufrufe im Chat starten jetzt eingeklappt — weniger visuelle Ablenkung bei gleichzeitig besserer Übersicht über den Fortschritt.
- KI-Antworten erscheinen jetzt mit einer sanfteren Wort-für-Wort-Animation für ein angenehmeres Leseerlebnis.

## Bugfix

- Spaces sind jetzt stabiler und schneller — korrekte Speicherberechnung, schnellere Synchronisation und robustere Fehlerbehandlung.
- Die Kostenberechnung für das Modell Sonnet 4.5 wurde an die offiziellen Anthropic-Preise angepasst — der Ausgabepreis war zuvor zu niedrig kalkuliert.
- Der Artefakt-Editor lässt sich wieder korrekt scrollen — ein Fehler durch CSS-Änderungen wurde behoben.
- Bei Verbindungsabbrüchen gehen keine Nachrichten mehr verloren — die Wiederholung von Chat-Nachrichten funktioniert jetzt zuverlässig.
- Der weclapp-Connector lädt Dokumente jetzt zuverlässiger herunter und geht besser mit API-Ratenlimits um.

## Security

- Wenn die KI ein Artefakt erstellt, wird jetzt geprüft, ob der Nutzer tatsächlich Zugriff auf den Ziel-Space hat.

## Refactor

- Die internen Chat-Endpunkte wurden zusammengeführt und vereinfacht — keine doppelte Logik mehr.

## Infra

- Der Release-Branch wird jetzt automatisch bei jedem Produktions-Deployment aktualisiert.
