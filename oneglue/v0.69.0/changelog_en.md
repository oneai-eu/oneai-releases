# OneGlue v0.69.0 — Sign-in, guests & single sign-on

**Release Date:** 2026-05-28

How people get into a tool that holds their most sensitive information is a security decision, not just a convenience one. OneGlue is built around passwordless, phishing-resistant sign-in from the ground up — and pairs it with controlled, time-limited access for the outsiders you occasionally need to bring in.

## Highlights

- **Passkey-only sign-in.** OneGlue uses Passkeys (WebAuthn) — there are no passwords to phish, reuse, or leak, because there are no passwords at all. You sign in with your device's biometrics or a security key, which is both more secure and genuinely faster.
- **Microsoft single sign-on.** Log in with your existing Microsoft Entra ID account via SSO, with guided setup for administrators, so your team uses the same trusted identity they already have instead of managing yet another login.
- **Time-limited guest access.** Invite external collaborators with access that can be scoped to just what they need and set to expire automatically — so temporary access really is temporary, and you never have to remember to revoke it later.
