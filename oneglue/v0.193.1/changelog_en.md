# OneGlue v0.193.1 — What's New

**Release Date:** 2026-07-10

A richer VM Fleet — with real cloud capacity and per-VM details.

## Every cloud at a glance

The VM Fleet now shows each private cloud as its own card. Each card headlines the cloud's **capacity** — CPU, memory and storage limits — alongside **current usage** and a running/stopped tally, so you can see how much of a cloud is in use without leaving the page. Clouds that are provisioned without a fixed limit are shown as **unlimited** rather than a misleading number.

## Click a VM for the details

The VM list is now cleaner — each row shows the essentials (status, name, CPU/RAM, primary IP). Click any virtual machine to open a detail panel with its full specifications: status, operating system, CPU and memory, host and all IP addresses.

## Coming soon: power controls

Start, Stop and Restart controls now appear in the VM detail panel as a preview. They're not active yet — the ability to power VMs on and off directly from OneGlue arrives in an upcoming release.
