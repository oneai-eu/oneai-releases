# OneGlue v0.193.1 — Neuerungen

**Veröffentlicht am:** 2026-07-10

Eine umfassendere VM-Flotte — mit echten Cloud-Kapazitäten und Details pro VM.

## Jede Cloud auf einen Blick

Die VM-Flotte zeigt jede Private Cloud jetzt als eigene Karte. Jede Karte nennt oben die **Kapazität** der Cloud — CPU-, Arbeitsspeicher- und Speicher-Limits — zusammen mit der **aktuellen Auslastung** und einer Zählung laufender/gestoppter VMs. So sehen Sie die Auslastung einer Cloud, ohne die Seite zu verlassen. Clouds ohne festes Limit werden als **unbegrenzt** angezeigt statt mit einer irreführenden Zahl.

## Für Details auf eine VM klicken

Die VM-Liste ist jetzt aufgeräumter — jede Zeile zeigt das Wesentliche (Status, Name, CPU/RAM, primäre IP). Klicken Sie auf eine virtuelle Maschine, um ein Detailfenster mit allen technischen Daten zu öffnen: Status, Betriebssystem, CPU und Arbeitsspeicher, Host und alle IP-Adressen.

## Demnächst: Steuerung des Betriebszustands

Schaltflächen für Start, Stopp und Neustart erscheinen jetzt als Vorschau im VM-Detailfenster. Sie sind noch nicht aktiv — die Möglichkeit, VMs direkt aus OneGlue heraus ein- und auszuschalten, kommt in einer der nächsten Versionen.
