OneGlue v0.198.0 makes remote RDP and VNC sessions far smoother: special characters like `|`, `\`, `@`, `{`, `[`, `]` and umlauts now type correctly, you can pick the keyboard layout per device (German by default), and the clipboard finally works in both directions (paste in, copy back out). The VM Fleet also gets a fresher feel — an on-demand Sync button per cloud, automatic refresh after Start/Stop/Restart, and an honest "last synced" time that shows when the inventory was last checked.
