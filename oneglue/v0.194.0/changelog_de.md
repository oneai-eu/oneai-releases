# OneGlue v0.194.0 — Neuerungen

**Veröffentlicht am:** 2026-07-11

Sehen Sie, wo jede VM läuft — und ob ihre Replikation gesund ist.

## Host, Cluster und Replikation auf einen Blick

Die VM-Flotte zeigt jetzt für jede virtuelle Maschine, **auf welchem Host und Failover-Cluster sie läuft** — direkt in der Übersicht, ohne Aufklappen. Daneben zeigt ein **Replikations-Status-Badge** auf einen Blick, ob die Hyper-V-Replikation einer VM gesund ist (grün), Aufmerksamkeit braucht (gelb) oder kritisch ist (rot).

## Vollständige Replikations-Details per Klick

Öffnen Sie eine VM, um das komplette Replikationsbild zu sehen: die **Rolle** (Primary oder Recovery), die **Health** und den aktuellen **Status**. Da replizierte VMs an zwei Orten existieren, erscheinen sie jetzt als zwei Einträge — das Primary und seine Recovery-Kopie — jeweils klar gekennzeichnet, sodass Sie Ihre Geo-Redundanz sehen (z.B. eine VM primär in einem Rechenzentrum mit ihrer Recovery-Kopie in einem anderen).
