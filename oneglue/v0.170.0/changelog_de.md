# OneGlue v0.170.0 — Was ist neu

**Release-Datum:** 2026-07-07

Die Verwaltung vieler Geräte und Organisationen geht jetzt schneller.

## Schnellere Geräte- & Organisations-Auswahl

Überall, wo Sie eine Organisation oder ein Gerät auswählen — beim Ausrollen eines Geräte-Agents, beim Verschieben eines Agents in eine andere Organisation oder beim Verknüpfen eines Agents mit seinem Gerät — ist das Dropdown jetzt ein **Suchfeld**: einfach tippen, und die passenden Einträge erscheinen sofort. Kein Scrollen mehr durch Hunderte von Einträgen. Ein echter Zeitgewinn in größeren Umgebungen.

## Intelligentere Geräte-Agent-Zuordnung

Wenn Sie einen OneGlue-Geräte-Agent ausrollen, verknüpft er sich jetzt automatisch mit dem richtigen Gerät, sobald *sowohl* Hostname *als auch* IP-Adresse übereinstimmen — ohne manuellen Schritt. Ist die Zuordnung nicht eindeutig, erhalten Sie ein Ein-Klick-**„Erkannte Zuordnung bestätigen"**, statt eine Liste durchsuchen zu müssen.
