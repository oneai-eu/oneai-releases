# OneGlue v0.189.1 — What's New

**Release Date:** 2026-07-10

Cleaner device-plugin management and reliable SCVMM / Hyper-V inventory sync.

## Cleaner device-plugin management

When you replace or remove a device plugin, the old version is now cleanly cleared from the device overview — loading a newer version automatically supersedes the previous one, so only the current plugin stays visible.

## SCVMM / Hyper-V inventory now connects reliably

Fixed a connection issue in the SCVMM provider so virtual-machine inventory syncs correctly.
