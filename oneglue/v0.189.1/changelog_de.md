# OneGlue v0.189.1 — Was ist neu

**Release-Datum:** 2026-07-10

Übersichtlicheres Geräte-Plugin-Management und zuverlässige SCVMM / Hyper-V-Inventarsynchronisierung.

## Übersichtlicheres Geräte-Plugin-Management

Wenn Sie ein Geräte-Plugin ersetzen oder entfernen, wird die alte Version jetzt sauber aus der Geräteübersicht entfernt — das Laden einer neueren Version überschreibt die vorherige automatisch, sodass nur das aktuelle Plugin sichtbar bleibt.

## SCVMM / Hyper-V-Inventar jetzt zuverlässig verbunden

Ein Verbindungsproblem im SCVMM-Provider wurde behoben, sodass das Inventar virtueller Maschinen korrekt synchronisiert wird.
