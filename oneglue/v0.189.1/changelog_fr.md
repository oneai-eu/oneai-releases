# OneGlue v0.189.1 — Nouveautés

**Date de sortie :** 2026-07-10

Gestion des plugins d'appareils plus claire et synchronisation de l'inventaire SCVMM / Hyper-V désormais fiable.

## Gestion des plugins d'appareils plus claire

Lorsque vous remplacez ou supprimez un plugin d'appareil, l'ancienne version est maintenant proprement effacée de la vue d'ensemble de l'appareil — le chargement d'une version plus récente remplace automatiquement la précédente, de sorte que seul le plugin actuel reste visible.

## L'inventaire SCVMM / Hyper-V se connecte désormais de manière fiable

Un problème de connexion dans le fournisseur SCVMM a été corrigé afin que l'inventaire des machines virtuelles se synchronise correctement.
