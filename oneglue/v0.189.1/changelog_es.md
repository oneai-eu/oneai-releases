# OneGlue v0.189.1 — Novedades

**Fecha de lanzamiento:** 2026-07-10

Gestión de plugins de dispositivo más clara y sincronización de inventario SCVMM / Hyper-V ahora fiable.

## Gestión de plugins de dispositivo más clara

Cuando reemplaza o elimina un plugin de dispositivo, la versión anterior se elimina limpiamente de la vista general del dispositivo — cargar una versión más reciente sustituye automáticamente a la anterior, de modo que solo el plugin actual permanece visible.

## El inventario SCVMM / Hyper-V ahora se conecta de forma fiable

Se ha corregido un problema de conexión en el proveedor SCVMM para que el inventario de máquinas virtuales se sincronice correctamente.
