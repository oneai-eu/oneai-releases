OneGlue v0.165.0 lays the groundwork for device plugins: the OneGlue agent can now run small, cryptographically signed, sandboxed plugins that a Tenant-Admin provisions straight from the interface — no server access needed. The first plugin is a read-only reader for onestack GPU nodes that surfaces GPU, inference-engine and model details plus live telemetry (VRAM, utilization, temperature, engine health) right on the device's page, and writes the stable facts to a searchable "AI Compute Node" record linked to the device. The reader is strictly read-only and never sees any secrets.
