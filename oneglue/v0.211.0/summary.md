OneGlue v0.211.0 adds a per-cloud VM inventory export: a new "Export CSV" action in the VM Fleet view downloads all virtual machines in a cloud as a CSV, including vCPU, RAM, storage, tags, and SCVMM custom properties (e.g. customer number, article code), with a summary totals row — ready for billing and reporting. Custom property columns can be given readable names per SCVMM server. The export is org-confined, contains no secret values, and requires no separate permission beyond the existing fleet-view access.
