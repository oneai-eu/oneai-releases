# OneGlue v0.172.0 — Nouveautés

**Date de sortie :** 2026-07-08

Un statut d'appareil plus clair et des agents d'appareil Windows plus stables.

## Statut de l'appareil clair en un coup d'œil

Sur la page d'un appareil, le statut de l'agent est désormais sans ambiguïté. Une fois qu'un agent d'appareil est connecté, OneGlue n'affiche plus d'invite résiduelle « enrôler l'agent » à côté du statut vert « agent connecté », et les actions en un clic de l'appareil apparaissent immédiatement au lieu d'être masquées derrière cette invite. Le statut des identifiants worker indique maintenant « Géré par l'agent » sur les hôtes avec agent connecté — il est donc clair qu'aucune configuration supplémentaire n'est nécessaire.

## Agents d'appareil Windows plus stables

Les agents d'appareil Windows restent connectés de façon plus fiable : un cas rare où un agent Windows pouvait se déconnecter et nécessiter un redémarrage manuel a été supprimé. Vos appareils Windows continuent de remonter leurs informations sans cette interruption.

## Sélecteurs plus rapides dans la configuration des intégrations

Davantage de sélecteurs d'appareil et d'organisation utilisés lors de la configuration des intégrations sont désormais des **champs de recherche** instantanés : commencez à taper pour trouver le bon appareil ou la bonne organisation au lieu de faire défiler une longue liste.
