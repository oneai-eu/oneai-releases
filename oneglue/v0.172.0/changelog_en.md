# OneGlue v0.172.0 — What's New

**Release Date:** 2026-07-08

Clearer device status and steadier Windows device agents.

## Clearer device status at a glance

On a device's page, the agent status is now unambiguous. Once a device agent is connected, OneGlue no longer shows a leftover "enroll the agent" prompt next to the green "agent connected" state, and the device's one-click actions appear right away instead of being hidden behind that prompt. Worker-credential status now reads "Agent-managed" on agent-connected hosts, so it's clear no extra setup is needed.

## Steadier Windows device agents

Windows device agents stay connected more reliably: a rare case where a Windows agent could go offline and need a manual restart has been removed. Your Windows devices keep reporting without that interruption.

## Faster pickers in integration setup

More of the device and organization pickers used when setting up integrations are now instant **search boxes** — start typing to find the right device or organization instead of scrolling a long list.
