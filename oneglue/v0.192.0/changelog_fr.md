# OneGlue v0.192.0 — Nouveautés

**Date de sortie :** 2026-07-10

La flotte de VM de votre organisation, désormais visible.

## Consultez les machines virtuelles de votre organisation

Les membres d'une organisation peuvent maintenant voir les machines virtuelles SCVMM appartenant à leur organisation — directement au sein de l'organisation, en lecture seule. Une nouvelle page Flotte de VM liste les clouds privés de l'organisation et les VM qu'ils contiennent (nom, statut, système d'exploitation, CPU/RAM, hôte, adresses IP), regroupées par cloud et consultables par recherche. L'entrée Flotte de VM n'apparaît dans le menu d'une organisation que lorsque des SCVMM clouds lui sont associées — elle reste invisible pour les organisations qui n'utilisent pas SCVMM.

Chaque organisation voit uniquement les VM de ses propres clouds mappées — aucune donnée d'autres organisations.
