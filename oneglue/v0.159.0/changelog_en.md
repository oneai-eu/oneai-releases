# OneGlue v0.159.0 — What's New

**Release Date:** 2026-07-03

A small, focused polish to the AI assistant experience — so you always know where you stand before you start typing.

## A clearer path to your AI assistant

The AI assistant needs your OneGlue workspace to be connected to oneAI before it can answer. Until now, a workspace that wasn't connected yet still showed an open chat box — you could type a full question, only to be told afterwards that it couldn't be answered. That was a confusing dead end, so we made the prerequisite obvious from the start.

- **Know before you type.** If your workspace isn't connected to oneAI yet, the assistant now says so clearly and up front — right where you'd begin chatting, on both the quick assistant button and the full assistant page — with a short explanation of what's needed.
- **One click to get set up.** The message links you straight to the connection settings, so there's an obvious next step instead of a message that goes nowhere.
- **Always discoverable.** The assistant stays visible for everyone, so you never lose sight of the feature — this simply makes the one-time connection step clear before you invest any typing.

_This release also includes ongoing behind-the-scenes improvements to how we build and ship OneGlue updates, keeping deployments reliable and consistent._
